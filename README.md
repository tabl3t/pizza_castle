# pizza_castle
🍕 Built a full-stack pizza ordering system from scratch for my DBMS course!
PizzaCastle is a role-based web application built with PHP and MySQL — no frameworks, just fundamentals.
#What it does:

Customers can browse the menu,
add items to cart,
place orders
Admins can manage the menu (add/remove pizzas) and update order statuses in real time
Full CRUD operations on a relational MySQL database

#Tech used: PHP · MySQL · HTML · CSS · Apache (XAMPP)
Key concepts applied:

Session-based authentication with role separation (admin vs customer)
Relational database design with 4 tables and JOIN queries
Soft delete pattern to preserve historical data
Password hashing with PHP's password_hash()

Still learning, but proud of how it came together! 🚀
